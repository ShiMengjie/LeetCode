    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }


    @Override
    public String toString() {
        return preorderTraversal(this).toString();
    }


    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> list = new LinkedList<>();
        pre(list, root);
        return list;
    }

    private void pre(List<Integer> list, TreeNode root) {
        if (root != null) {
            // 遍历根节点
            list.add(root.val);
            // 遍历左子树
            pre(list, root.left);
            // 遍历右子树
            pre(list, root.right);
        }
    }
}

## [58. 最后一个单词的长度](https://leetcode-cn.com/problems/length-of-last-word/)

> 2021-09-21

```java
// 双指针解法
class Solution {
    public int lengthOfLastWord(String s) {
        int n = s.length(), idx1 = n - 1, idx2 = n - 1;
        while (s.charAt(idx1) == ' ') {
            idx1--;
            idx2--;
        }
        while (idx2 >= 0 && Character.isLetter(s.charAt(idx2))) {
            idx2--;
        }
        return idx1 - idx2;
    }
}

// 单指针解法
class Solution {
    public int lengthOfLastWord(String s) {
        int count = 0;
        for (int i = s.length() - 1; i >= 0; i--) {
            if (s.charAt(i) != ' ') {
                count++;
            } else if (s.charAt(i) == ' ' && count != 0) {
                return count;
            }
        }
        return count;
    }
}
```

典型的双指针问题，一个指向单次单词最后一个字符，一个指向单词前的空格，用两个指针相减，就是单词长度。

也可以使用单指针来做，需要额外记录单词长度。

## [725. 分隔链表](https://leetcode-cn.com/problems/split-linked-list-in-parts/)

> 2021-09-22

```java
class Solution {
    public ListNode[] splitListToParts(ListNode head, int k) {
        // len : 链表长度，begin : 指向头结点
        int len = 0;
        ListNode begin = head;
        while (head != null) {
            len++;
            head = head.next;
        }
        // 如果链表长度为 0，直接返回结果
        if (len == 0) {
            return new ListNode[k];
        }
        // part : res 每个元素的节点数
        // remain : 剩余的节点数，前 remain 个元素，每个元素有 part + 1 个节点
        int part = len / k, remain = len % k;
        // part == 0 : 节点数小于 len，前 remain 个元素每个元素只有一个节点
        ListNode[] res = new ListNode[k];
        if (part == 0) {
            for (int i = 0; i < remain; i++) {
                res[i] = begin;
                begin = begin.next;
                res[i].next = null;
            }
        } else {
            ListNode tmp;
            for (int i = 0; i < k; i++) {
                res[i] = begin;
                // 前 remain 个元素每个元素有 part + 1 节点，其他元素有 part 个节点
                if (i < remain) {
                    // 遍历 part + 1 个节点
                    for (int j = 0; j <= part; j++) {
                        if (j == part) {
                            tmp = begin;
                            begin = begin.next;
                            tmp.next = null;
                        } else {
                            begin = begin.next;
                        }
                    }
                } else {
                    // 遍历 part 个节点
                    for (int j = 0; j < part; j++) {
                        if (j == part - 1) {
                            tmp = begin;
                            begin = begin.next;
                            tmp.next = null;
                        } else {
                            begin = begin.next;
                        }
                    }
                }
            }
        }
        return res;
    }
}
```

简单的暴力解法，关键在于分成k段，确定每段里节点的个数。

测试用例：head 有可能为空，要返回一个大小为k的空数组。

## [326. 3的幂](https://leetcode-cn.com/problems/power-of-three/)

> 2021-09-23

递归地除以3:，判断是否有余数、系数是否为1

```java
class Solution {
   public boolean isPowerOfThree(int n) {
        return isPower(n, 3);
    }

    public boolean isPower(int n, int k) {
        if (n < k && n != 1) {
            return false;
        }
        if (n == 1) {
            return true;
        }
        if (n % k != 0) {
            return false;
        }
        n = n / k;
        return isPower(n, k);
    }
}
```

## [430. 扁平化多级双向链表](https://leetcode-cn.com/problems/flatten-a-multilevel-doubly-linked-list/)

> 2021-09-24

使用二叉树前序遍历，child 看作是左子节点，next 看作是右子节点。

```java
/*
// Definition for a Node.
class Node {
    public int val;
    public Node prev;
    public Node next;
    public Node child;
};
*/

class Solution {
    public Node flatten(Node head) {
        if(head == null) {
            return head;
        }
        
        List<Node> list = new LinkedList<>();
        list.add(head);

        Stack<Node> stack = new Stack<>();
        if (head.next != null) {
            stack.push(head.next);
        }
        if (head.child != null) {
            stack.push(head.child);
        }
        while (!stack.isEmpty()) {
            Node node = stack.pop();
            if (node.next != null) {
                stack.push(node.next);
            }
            if (node.child != null) {
                stack.push(node.child);
            }
            list.add(node);
        }

        list.get(0).prev = null;
        for (int i = 0; i < list.size() - 1; i++) {
            list.get(i).next = list.get(i + 1);
            list.get(i + 1).prev = list.get(i);
            list.get(i).child = null;
        }
        list.get(list.size() - 1).next = null;
        list.get(list.size() - 1).child = null;
        return list.get(0);
    }
}
```

## [583. 两个字符串的删除操作](https://leetcode-cn.com/problems/delete-operation-for-two-strings/)

> 2021-09-25

先求出两个字符串的最长公共子序列LCS 的长度 L，要做的删减次数就是两个字符串的长度 M+ N - L - L。

```java
class Solution {
   public int minDistance(String word1, String word2) {
        int m = word1.length(), n = word2.length();
        int[][] dp = new int[m][n];

        if (word1.charAt(0) == word2.charAt(0)) {
            dp[0][0] = 1;
        }
        for (int i = 1; i < m; i++) {
            if (word1.charAt(i) == word2.charAt(0)) {
                dp[i][0] = 1;
            } else {
                dp[i][0] = dp[i - 1][0];
            }
        }
        for (int j = 1; j < n; j++) {
            if (word1.charAt(0) == word2.charAt(j)) {
                dp[0][j] = 1;
            } else {
                dp[0][j] = dp[0][j - 1];
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (word1.charAt(i) == word2.charAt(j)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        return m + n - 2 * dp[m - 1][n - 1];
    }
}
```

class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int carry = (a & b) << 1;
            a = a ^ b;
            b = carry;
        }
        return a;
    }
}

## [639. 解码方法 II](https://leetcode-cn.com/problems/decode-ways-ii/)

> 2021-09-27

比较复杂的动态规划，在寻找状态转移方程时，与 前一个字符、前两个字符 先关，找到多种组合情况。直接看官方[题解](https://leetcode-cn.com/problems/decode-ways-ii/solution/jie-ma-fang-fa-ii-by-leetcode-solution-23af/)，自己再思考一遍，基本就懂了。

这种类型的题目，关键在于找到，当前字符与前面字符的组合情况。

```java
class Solution {
    static final int MOD = 1000000007;

    public int numDecodings(String s) {
        int n = s.length();
        // a = f[i-2], b = f[i-1], c = f[i]
        long a = 0, b = 1, c = 0;
        for (int i = 1; i <= n; ++i) {
            c = b * check1digit(s.charAt(i - 1)) % MOD;
            if (i > 1) {
                c = (c + a * check2digits(s.charAt(i - 2), s.charAt(i - 1))) % MOD;
            }
            a = b;
            b = c;
        }
        return (int) c;
    }

    public int check1digit(char ch) {
        if (ch == '0') {
            return 0;
        }
        return ch == '*' ? 9 : 1;
    }

    public int check2digits(char c0, char c1) {
        if (c0 == '*' && c1 == '*') {
            return 15;
        }
        if (c0 == '*') {
            return c1 <= '6' ? 2 : 1;
        }
        if (c1 == '*') {
            if (c0 == '1') {
                return 9;
            }
            if (c0 == '2') {
                return 6;
            }
            return 0;
        }
        return (c0 != '0' && (c0 - '0') * 10 + (c1 - '0') <= 26) ? 1 : 0;
    }
}
```

## [437. 路径总和 III](https://leetcode-cn.com/problems/path-sum-iii/)

> 2021-09-28

暴力求解，找到所有的路径。

```java
class Solution {
    public int pathSum(TreeNode root, int targetSum) {
        if (root == null) {
            return 0;
        }

        int ret = rootSum(root, targetSum);
        ret += pathSum(root.left, targetSum);
        ret += pathSum(root.right, targetSum);
        return ret;
    }

    public int rootSum(TreeNode root, int targetSum) {
        int ret = 0;

        if (root == null) {
            return 0;
        }
        int val = root.val;
        if (val == targetSum) {
            ret++;
        }

        ret += rootSum(root.left, targetSum - val);
        ret += rootSum(root.right, targetSum - val);
        return ret;
    }
}
```
