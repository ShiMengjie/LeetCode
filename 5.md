# [5. 最长回文子串](https://leetcode-cn.com/problems/longest-palindromic-substring/)

> 给你一个字符串 `s`，找到 `s` 中最长的回文子串。

**示例1**

```java
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

**示例2**

```java
输入：s = "cbbd"
输出："bb"
```

**示例 3**

```java
输入：s = "a"
输出："a"
```

**示例 4**

```java
输入：s = "ac"
输出："a"
```

## 分析阶段



### 编码阶段

```java
class Solution {
    public String longestPalindrome(String s) {
        int N = s.length();
        // 特殊情况
        if (N == 0 || N == 1) {
            return s;
        }
        // 数组，记录是否有
        boolean[][] dp = new boolean[N][N];

        int start = 0, end = 0, l = 0;
        for (int i = N - 1; i >= 0; i--) {
            for (int j = i; j < N; j++) {
                if (j == i) {
                    dp[i][j] = true;
                } else {
                    boolean b = s.charAt(i) == s.charAt(j);
                    if (j == i + 1) {
                        dp[i][j] = b;
                    } else {
                        dp[i][j] = dp[i + 1][j - 1] && b;
                    }
                }
                // 更新回文子串的下标
                if (dp[i][j] && j - i + 1 > l) {
                    start = i;
                    end = j;
                    l = j - i + 1;
                }
            }
        }
        return s.substring(start, end + 1);
    }
}
```
